# Real Estate Data Analysis – IBM Guided Project

This project was completed as part of IBM’s *Data Analyst Professional Certificate* on Coursera.  
The goal was to analyze real estate data using Python to uncover key insights and trends influencing housing prices.

## Key Highlights
- Cleaned and explored real estate data using **Pandas** and **Matplotlib**
- Performed **exploratory data analysis (EDA)** to find key correlations between price, size, and location
- Created **data visualizations** for better insight presentation
- Translated analytical findings into **business recommendations**

## Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Jupyter Notebook
